## AES固件加密工具
### 项目：AES文件加密工具
### AES模式：CTR
### 开发软件：java(swing)
### 开发环境：
*  `Windows 10(myeclipse 2016)`
*  `Ubuntu 16.04`
### jdk版本：
*  `1.8.0_131(Windows)`
*  `1.8.0_121(Linux)`
### 编译：
####  Windows:
     myeclipse -> import -> ...
####  Linux:
     进入工程目录
       make
     另
       make = make all     仅编译
       make jar            编译并打包
       make launch         编译并运行
       make all jar launch 编译、打包、运行
####  Mac OS:
      不出意外,应该是支持的,用make编译
